**a. Consigna:** Dibujar el árbol de expresión para la inicialización de los contadores:
~~~
nl = nw = nc = 0.  
~~~  
  
Podemos observar que la notación es infija (simétrica), por lo tanto dibujamos el árbol de expresión en base a eso:
